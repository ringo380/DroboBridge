# Documentation

This directory contains documentation assets for DroboBridge.

## Images

The `images/` directory should contain:

- `screenshot-placeholder.png` - Main app screenshot for README (replace with actual screenshot)
- Additional screenshots showing:
  - Overview tab with storage visualization
  - Volumes tab with mount options
  - File browser in action
  - Diagnostics export

## Adding Screenshots

1. Take screenshots at 2x resolution (Retina)
2. Crop to show relevant UI only
3. Use PNG format for best quality
4. Name files descriptively: `feature-name-description.png`

## Screenshot Guidelines

- Show the app in a realistic state (with Drobo connected if possible)
- Blur or remove any personal data
- Use light mode for primary screenshots
- Consider adding dark mode variants
